# Architecture Decision Records (ADR)

Short, dated records of significant technical decisions for VoiceType.

**Format:** `YYYY-MM-DD-<slug>.md` or `<topic>.md` for ongoing-living documents.

**Public vs private:**
- Public ADRs live directly in this directory — shareable with contributors.
- Private or work-in-progress notes go in `docs/decisions/private/` (gitignored).

**Template:**

```markdown
# <title>

- Status: PROPOSED | ACCEPTED | SUPERSEDED
- Date: YYYY-MM-DD

## Context

_Why are we making this decision now?_

## Decision

_What did we decide? Be specific._

## Consequences

_What does this unlock, block, or require?_

## Alternatives considered

_2-3 other options and why they were rejected._
```
